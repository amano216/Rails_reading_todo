# Rails読解力向上クイズ

Ruby on Railsアプリケーションのコードを読む力を養うための100問クイズです。

## 対象者
- Rails初心者
- コードリーディング力を向上させたい方
- 体系的にRailsを学習したい方

## 内容

### 初級編（50問）
- Ruby基本構文
- Railsディレクトリ構造
- ルーティングの基礎
- MVCパターンの理解

### 中級編（35問）
- ActiveRecordとデータベース操作
- ヘルパーメソッド
- ビューの仕組み

### 上級編（15問）
- メタプログラミング
- Rails内部機構
- パフォーマンス最適化
- セキュリティ対策

## 使い方
1. [rails_reading_quiz.md](rails_reading_quiz.md)を開く
2. 初級編から順番に解いていく
3. 答えを確認し、理解できない部分は実際にコードを書いて確認

## 学習のポイント
- 間違えた問題は実際にコードを書いて動作確認することが重要
- 初級→中級→上級と段階的に進める
- 繰り返し解いて、パターンを身につける
